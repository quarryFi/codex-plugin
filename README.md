# quarryFi Codex Plugin

quarryFi time tracking plugin for OpenAI Codex. Tracks R&D activity for automated tax credit documentation.

## Setup

```bash
curl -fsSL https://raw.githubusercontent.com/quarryFi/codex-plugin/main/setup.sh | bash
```

You'll be prompted for your QuarryFi API key. Get one from your [QuarryFi dashboard](https://quarryfi.smashedstudiosllc.workers.dev/dashboard).

## What it tracks

- Session start/end times
- Project context
- Language and file types
- Branch information

All data is sent to your QuarryFi account for R&D tax credit qualification analysis.
